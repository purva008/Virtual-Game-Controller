AI Virtual Game Controller

AI Virtual Game Controller is an intelligent Human–Computer
Interaction (HCI) system developed as a Final Year Engineering
Project. The application enables users to control keyboard-based
games using real-time hand gestures captured through a standard
web camera, eliminating the need for conventional input devices.

The system combines Computer Vision, Artificial Intelligence,
and Machine Learning techniques to deliver a robust, accurate,
and responsive gesture recognition framework capable of providing
a seamless gaming experience.

Project Objective
------------------------------------------------------------

The primary objective of this project is to demonstrate the
practical implementation of Artificial Intelligence and Computer
Vision techniques for natural human-computer interaction. The
system replaces traditional keyboard controls with intuitive hand
gestures while maintaining high accuracy, stability, and real-time
performance suitable for interactive gaming applications.
