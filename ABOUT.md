# About the Project

AI Virtual Game Controller is a Computer Vision application developed using Python, OpenCV, MediaPipe, and PyAutoGUI.

The system captures live video from a webcam, detects hand landmarks, recognizes predefined gestures, and converts them into mouse actions such as cursor movement, clicking, dragging, and scrolling.

The project demonstrates the practical use of Human-Computer Interaction (HCI), Computer Vision, and Automation.
